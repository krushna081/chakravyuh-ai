## Features

- **Agent Marketplace** — Community agent registry with versioning and ratings
- **Knowledge Graph** — Entity extraction, graph-based reasoning, visual explorer
- **Memory Consolidation** — Auto-summarization, forgetting curves, cross-agent sharing
- **Multi-Agent Debate** — Structured arguments, judge agent, balanced conclusions
- **Agent Training** — Collect interaction data, fine-tune, RLHF
- **Edge Deployment** — Lightweight agents on edge devices, offline-first
