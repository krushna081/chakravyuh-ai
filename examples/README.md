# Examples

> Usage examples and tutorials for Chakravyuh AI.

---

## Available Examples

| Example | Description | File |
|---------|-------------|------|
| Basic Chat | Send a message to an agent | `basic-chat.mjs` |
| Custom Agent | Create and register a custom agent | `custom-agent.mjs` |
| Workflow | Define and execute a multi-step workflow | `workflow.mjs` |
| MCP Server | Create a custom MCP server | `mcp-server.mjs` |
| Provider Config | Multiple provider configuration | `provider-config.yaml` |
| Memory Query | Query and search agent memory | `memory-query.mjs` |

---

## Running Examples

```bash
# Ensure Chakravyuh is running
npm run dev

# Run a specific example
node examples/basic-chat.mjs
```
